# To-Do-Web-App
Python To-Do-App using Flask and SQLite.

## Launch the app
1. Clone git repository.
2. Open the repository folder and launch the app using flask. 
`$ flask run`
3. Open the flask app link in your browser.
4. Enjoy the app!
